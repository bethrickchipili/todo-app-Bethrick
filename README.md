HOW TO RUN THE APP  
Step One: Make sure you have an IDE and necessary extentions(if needed) installed

Step Two: Open the App in the IDE

Step Three: Run the App

Step Four: Choose the number corresponding with what you want to see from the menu:

1. To add a new Task
2. To view tasks
3. To mark a task as complete
4. To delete a task
5. To exit
